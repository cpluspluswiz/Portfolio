My Projects

Block Breakers

Blockbreakers is a fun, arcade type game that lets you break blocks using a ball that bounces off a paddle. The objective is to break all the blocks to complete the level. This was made using the Unity engine and was programmed in C#.

First To A Hundred

First to hundred is a 2 player game where the player adds score to his/her round by rolling the dice, but if the player rolls a 1 or two sixs in a row then the player looses all of their score that he/she attained in that round and the player switches turns. The player can choose to "hold" which will add the player's round score to their overall score. This game was made using HTML, CSS, and Javascript.

Todo List

A todo list application that lets users create their own list of tasks that lets them plan and prioritize their day. The design and layout of this application was made using HTML and CSS while the backend was done using Javascript and nodejs. The list was stored in a mongodb database.
